# 📊 Sales Analytics Dashboard Using Google Sheets

An interactive **Sales Analytics Dashboard** built using **Python (Streamlit)** and **JavaScript (Chart.js)** to perform **Exploratory Data Analysis (EDA)** on CSV datasets and sales data.

This project enables users to upload datasets, apply filters, and visualize insights through dynamic charts — all without writing code.

---

## 🚀 Project Overview

The **Sales Analytics Dashboard** helps businesses and analysts quickly explore and understand sales data. It provides:

* 📊 Interactive visualizations
* 📈 KPI tracking
* 🎛 Dynamic filtering
* 🔍 Real-time data insights

---

## ✨ Features

### 📂 Data Upload

* Upload CSV datasets
* Data processing using **Pandas**

### 🔎 Data Preview

* View dataset in table format
* Statistical summary (mean, min, max, etc.)

### 🎛 Interactive Filters

* Sidebar column selection
* Range slider for numeric filtering
* Real-time updates

### 📊 Visualizations

The dashboard supports multiple chart types:

* Histogram
* Box Plot
* Bar Chart
* Pie Chart
* Heatmap (Correlation Matrix)
* Line Chart
* Radar Chart
* Polar Area Chart
* Doughnut Chart
* 3D Scatter Plot

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/sales-analytics-dashboard.git
cd sales-analytics-dashboard
```

### 2️⃣ Install Required Libraries

```bash
pip install streamlit pandas matplotlib seaborn plotly
```

### 3️⃣ Run the Streamlit App

```bash
streamlit run app.py
```

### 4️⃣ Open in Browser

```
http://localhost:8501
```
